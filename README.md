# Clever Snake

Игра Clever Snake с тремя режимами игры и многоязычной поддержкой (русский, английский, корейский).

## Описание

Это современная версия классической игры "Змейка" с образовательными элементами. Игра включает три режима:

1. **Классическая змейка** - традиционная игра без образовательной цели
2. **Викторина** - змейка с всплывающими тестовыми заданиями
3. **Змейка со словами** - змейка, собирающая буквы для составления слов

## Требования

- Python 3.7+
- Pygame 2.0+
- Корейские шрифты (для отображения корейских символов)

### Установка корейских шрифтов

Для корректного отображения корейских символов в игре необходимо установить корейские шрифты:

- **Windows**: Установите Korean Language Pack через Settings → Language
- **macOS**: Добавьте корейский язык в System Preferences → Language & Region
- **Linux**: `sudo apt-get install fonts-nanum` (Ubuntu/Debian)

Подробные инструкции см. в файле [korean_fonts_setup.md](korean_fonts_setup.md)

## Установка

1. Убедитесь, что у вас установлен Python 3.7 или выше
2. Установите Pygame:
   ```bash
   pip install pygame
   ```
3. Скачайте файл `snake_game.py`
4. Запустите игру:
   ```bash
   python snake_game.py
   ```

## Управление

### Основные клавиши:
- **W** - Движение вверх
- **S** - Движение вниз  
- **A** - Движение влево
- **D** - Движение вправо
- **C** - Пауза
- **V** - Продолжить игру
- **Q** - Выход

### В главном меню:
- **1, 2, 3** - Выбор режима игры
- **S** - Настройки
- **Q** - Выход

### В настройках:
- **1, 2, 3** - Выбор языка интерфейса (русский, английский, корейский)
- **4, 5, 6** - Выбор языка игры (русский, английский, корейский)
- **7, 8, 9, 0, -** - Выбор разрешения экрана (800x600, 1000x700, 1200x800, 1366x768, 1920x1080)
- **ESC** - Назад в меню

### В викторине:
- **1, 2, 3, 4** - Выбор ответа
- **ESC** - Пропустить вопрос

### При окончании игры:
- **R** - Перезапустить
- **M** - В главное меню

## Режимы игры

### 1. Классическая змейка
- Обычная игра в змейку
- Собирайте красные яблоки для роста
- Избегайте столкновения с собственным телом

### 2. Викторина
- При поедании красного яблока появляется вопрос
- Выберите правильный ответ для получения очков
- Неправильный ответ не дает очков

### 3. Змейка со словами
- Собирайте красные яблоки с буквами
- Буквы появляются в правильном порядке для составления слова
- За каждое собранное слово получаете бонусные очки

## Особенности

- **Телепортация**: При выходе за границы экрана змейка появляется с противоположной стороны
- **Многоязычность**: Поддержка русского, английского и корейского языков
- **Образовательные элементы**: Тесты и сбор слов для развития навыков
- **Настройки разрешения**: Выбор из 5 популярных разрешений экрана
- **Кроссплатформенность**: Работает на Windows, macOS и Linux
